## 📖 About

1Tap Cleaner 5.2.2 is a lightweight Windows utility designed to help users free up disk space, remove temporary files, and optimize system performance with a single click. It focuses on clearing junk files, browser caches, and unnecessary system logs, making it a straightforward tool for maintaining a cleaner PC without complex configurations.

## ⚙️ Requirements

- Windows 7 or later (32-bit or 64-bit)
- 50 MB of free disk space
- Internet connection for initial download
- Administrator privileges recommended for full system cleaning
- .NET Framework 4.5 or higher (may be installed automatically if missing)

## ✨ Features

- **One-Click Cleanup** 🧹 — Automatically scans and removes temporary files, cache, and junk data from your system
- **Browser Cache Cleaner** 🌐 — Clears cookies, history, and temporary internet files from Chrome, Firefox, Edge, and other browsers
- **Disk Space Analyzer** 📊 — Quickly identifies large files and folders that can be safely removed
- **Registry Cleanup** ⚙️ — Scans and removes invalid or obsolete registry entries to improve stability
- **Startup Manager** 🚀 — Disable or enable startup programs to speed up boot time
- **System Restore Point Creator** 💾 — Optionally create a restore point before cleaning for safety
- **Quiet Mode** 🤫 — Run cleaning tasks silently without user interaction

## 🔧 Configuration

1Tap Cleaner 5.2.2 comes pre-configured with safe default settings. However, you can customize cleaning actions via the `settings.json` file located in the same directory as the executable.

Example configuration snippet:

```json
{
  "cleanBrowsers": true,
  "cleanTempFiles": true,
  "cleanRecycleBin": false,
  "cleanRegistry": false,
  "createRestorePoint": true,
  "quietMode": false
}
```

- Set `"cleanRegistry"` to `true` if you want registry cleaning (not recommended for beginners).
- Set `"quietMode"` to `true` to run without pop-up confirmations.

**Q: Is 1Tap Cleaner safe to use? Will it cause any issues?**  
A: Yes, it is safe when used with default settings. The tool is designed to clean only non-critical temporary files. However, we recommend using the built-in System Restore Point creator before cleaning if you are concerned about removing registry entries or other system data. There is no ban risk as this is a system utility, not a game cheat.

**Q: I get an error about missing DLLs or runtime components. What should I do?**  
A: This typically happens if .NET Framework is not installed. Run the installer as Administrator, and Windows should automatically prompt you to download the required runtime. If not, manually install .NET Framework 4.5+ from Microsoft's website.

**Q: Can I schedule automatic cleanups?**  
A: Yes, you can use Windows Task Scheduler to run the executable with the `--quiet` flag at your desired interval. For example, create a task that triggers on login or weekly.
